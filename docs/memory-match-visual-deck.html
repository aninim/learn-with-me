Slide 02 — Foundation
Why It Works: The Science
5 research principles baked into every design decision.
🧠
Working Memory Predicts Academic Readiness
WM strength at age 5 predicts reading and math outcomes better than IQ alone. Tablet-based WM games produce gains that persist 1+ month after play stops — genuine neural change.
Alloway (2012); University of Maryland RCT, n=148 kindergarteners (PubMed 2019)
📉
The Ebbinghaus Spacing Effect
Memory decays rapidly. 5 spaced encounters beat 10 massed encounters on the same day. Cards missed in round 1 get higher weight in round 2. That's Anki for 4-year-olds.
Ebbinghaus (1885); ResearchGate spaced-rep mobile learning meta-analysis (2014)
🔗
Associative Memory > Rote Memory
Matching א to 🦁 (Ari) creates a dual-coded memory trace — image + sound + letter, all at once. Traditional aleph-bet charts have used this for millennia. We digitize it.
Paivio Dual-Coding Theory; eeBoo Research (2025)
😌
Stress Blocks Memory Formation
Negative feedback triggers cortisol. Cortisol inhibits hippocampal encoding. Every wrong answer uses gentle audio + card flip — no red X, no buzzer, no punitive signal.
Lupien et al. (2009) — stress & hippocampus; positive reinforcement architecture
👁️
Subitizing & Gestalt Number Sense
Children recognize dot patterns without counting up to 5. Dice patterns, triangle clusters, and square groups bypass serial counting — building number sense at the perceptual level.
Dehaene (2011); Clements subitizing research; Gestalt proximity principle
🎮
250ms Mismatch Window = Passive Learning
Cards stay face-up for 250ms after a mismatch — long enough to register both images, short enough to feel snappy. A "loss" becomes a learning moment. Unique to this design.
HCI: Fitts (1954); cognitive load theory; this module's core UX innovation

Slide 06 — Card States

Slide 12 — Locked Decisions
🔒 What's Non-Negotiable.
These decisions are set. Codi does not deviate. Can only be re-opened by Oren.
🎴
M-01: Conceptual pairs (not identical) for ages 5+
Vocabulary reinforcement > pure pattern match. Identical pairs only for Band A (ages 3–4).
⏱️
M-02: No timers in any mode
Time pressure activates stress. Stress blocks hippocampal encoding. Children set their own pace.
👁️
M-03: 250ms face-up window on mismatch
Transforms a miss into a passive learning moment. Both cards visible before flip-back.
🆘
M-04: Auto-solve after 4 misses
No dead ends. Frustration is the enemy of memory formation. Warm voice cue, not a failure state.
⭐
M-05: Stars = celebration only
1-star = full celebration. Star loss never shown. Stars are never a punishment mechanism.
🤖
M-07: AI only on stuck state
Not on every miss. Not on every round. After 2 misses on same pair only. Cached 7 days.
📊
M-08: Spaced repetition across sessions
localStorage tracks per-pair accuracy. Weak pairs return sooner. Durable learning, not one-session entertainment.
🏆
M-09: Memory Match = canonical Small Win
Any module can call window.launchSmallWin() to reset confidence with 4-card auto-success.
🔲
M-10: Matched pairs stay visible (grayed)
Default locked. Reduces cognitive load of tracking remaining cards. Selectable in future version.
🚫
SCOPE: Ages 3–6 only
Band C (7–8) with math compositions, 4×6 grids, and challenge modes is out of scope. Backlog v1.5.
