Mood Recovery — Small Win
Trigger Conditions
CV detects frown / sadness
OR 2 consecutive wrong answers
OR touch idle >5 seconds
→ System enters "Small Win" mode
Design Intent
Never show "wrong" — reframe as a break
Confetti falls = celebration not pity
Return to main flow after 1 win
Tone: warm, upbeat, never condescending
Game cards = 90px tall ✅

Yellow — Stars / Accent
